Senior Design Project: OS Visualizations

Date of Completion: May 5th, 2021

Description: My senior design project. This is the last project that I will complete at SUNY New Paltz. Part of my graduation requirements this project is done in a CPS485: Project Group, I am working in group 6. We were tasked with constructing a website and Visualizations for Operating System algorithms, where as my specific algorithms are focused on Disk Scheduling. Here are some images and designs I created for this final project. The project is still in the works today and will be completed by the end of the Spring 2021 semester.

Link: OS Visualizations

Exploring the Solar System: Solar System Website

Semester of Completion: Fall 2020

Description: For my Astronomy minor I chose to make a website as a final project to my Exploring the Solar System class. This was done through different techniques in HTML5 / CSS and bootstrap to display different information I had learned about our solar system over the course of the semester. It also included sections of my own research on other topics such as telescopes and satelights. It was a lot of fun to make and a good mix between learning front-end Web development and Astronomy.

Link: Solar System Website

Web Programming: Social Media Exercise Application

Semester of Completion: Fall 2020

Description: This project was done for the Web Programming Capstone Elective and it featured mulitple new concepts in web development I was unfamiliar with. This was done through an MVC type framework where I worked with front-end and back-end development to create an Exercise App / social media platform, allowing users to keep track of friends exercises and post about there new ones.

Languages and Technologies: JavaScript, mySQL, Bulma, Vue.js, Buefy, VSC, mySQLfreehosting, PHPmyadmin, HTML5 / CSS3, Postman, GitHub, Heroku, and more...
